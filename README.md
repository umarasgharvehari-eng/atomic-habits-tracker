# ⚛️ Atomic Habits Tracker

A Streamlit app to build better habits using James Clear's Atomic Habits framework.

## Features
- ✅ Daily habit tracking with streaks
- 🎯 2-Minute Rule implementation
- 🔗 Habit stacking support
- 📊 Progress visualization
- 🎉 Celebration rewards
